## Simple static blog made with Next.js and MDX and TypeScript

Live demo: coming soon

This is simple example uising Next.js with support for

- TypeScript
- React
- Next.js
- MDX
- Styled Component
- Semantic UI
- Firebase

next plan

- react-redux
- redux-thunk

### How to setup?

```
# install dependencis
yarn install

# serve with hot reload at localhost:3000
yarn dev
```
